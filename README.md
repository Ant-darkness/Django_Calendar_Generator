# 📅 Django Calendar Generator

A simple and responsive **Calendar Generator** built with **Django** for the backend and **HTML + CSS** for the frontend. The application dynamically generates calendars based on user input and demonstrates Django fundamentals such as views, templates, routing, and date handling.

---

## 📌 Project Overview

The Django Calendar Generator is a lightweight web application that allows users to generate and display calendars dynamically. It serves as a practical project for learning Django development, template rendering, and working with Python's built-in calendar and datetime modules.

---

## 🚀 Features

* Generate monthly calendars dynamically
* Clean and responsive user interface
* Built using Django MVC architecture
* HTML and CSS frontend
* Easy to customize and extend

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/django-calendar-generator.git
```

### 2. Navigate into the project

```bash
cd django-calendar-generator
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run migrations

```bash
python manage.py migrate
```

### 6. Start the development server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
